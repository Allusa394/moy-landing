# Правила работы с проектом my-landing

## О проекте
Лендинг-портфолио цифрового мастера (сайты, приложения, AI-агенты).
Одностраничный сайт для привлечения клиентов.

## Файлы проекта
- `index.html` — лендинг (единственный файл сайта, всё внутри)
- `brief.md` — техническое задание (структура, дизайн, контент)
- `research.md` — исследование (конкуренты, тренды, экспертная оценка)

## Технические требования
- Один HTML-файл — стили, скрипты, SVG внутри
- Никаких внешних библиотек и CDN
- Адаптивная вёрстка mobile-first
- Загрузка < 3 секунд

## Дизайн
- Тёмный фон (#0a0a0b)
- Светлый читаемый текст (#e8e8ed) — не тусклый!
- Акцентный цвет — оранжевый (#ff8c00)
- Контраст текста WCAG AA (минимум 4.5:1)
- Glassmorphism-карточки, scroll-анимации, hover-эффекты
- Поддержка prefers-reduced-motion

## Структура лендинга (10 секций)
1. Hero (typing-эффект, нейронная сетка)
2. Цифры доверия (счётчики)
3. Услуги (3 карточки: сайты, приложения, AI)
4. Портфолио (4 кейса с tilt-эффектом)
5. Процесс работы (6 шагов)
6. Отзывы (3 карточки)
7. Обо мне (фото + текст)
8. Тарифы (3 пакета)
9. FAQ (7 вопросов, аккордеон)
10. Контакты (форма → Telegram + мессенджеры)
