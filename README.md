# Цифровой мастер — лендинг-портфолио

Одностраничный сайт-портфолио для цифрового специалиста, который создаёт сайты, приложения и AI-агентов.

## Что внутри

- Современный дизайн с тёмной темой и оранжевым акцентом
- 10 секций: hero, услуги, портфолио, процесс, отзывы, тарифы, FAQ и другие
- Плавные анимации при прокрутке
- Форма заявки с отправкой в Telegram
- Полная адаптивность для мобильных устройств
- Один HTML-файл, без внешних библиотек

## Технологии

HTML + CSS + JavaScript — всё в одном файле, ноль зависимостей.

## Как открыть

Скачайте `index.html` и откройте в браузере.
