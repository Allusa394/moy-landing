## 1. О проекте

### 1.1 Цель сайта

Одностраничный лендинг-портфолио, который:
- Привлекает клиентов на услуги (сайты, приложения, AI-агенты)
- Доказывает экспертизу через кейсы с результатами
- Конвертирует посетителя в заявку (целевая конверсия: 5-10%)
- Работает как «визитка + продавец 24/7»

### 1.2 Целевая аудитория

| Сегмент | Кто это | Что ищет | Как принимает решение |
|---------|---------|----------|----------------------|
| **Основной** | Владелец малого/среднего бизнеса | Сайт или приложение для бизнеса | Смотрит портфолио, цены, отзывы |
| **Растущий** | Предприниматель, стартапер | AI-агент для автоматизации | Ищет конкретные кейсы с AI |
| **Дополнительный** | Маркетолог, менеджер проекта | Исполнитель для задачи от руководства | Важны сроки, процесс, договор |

**Общий портрет:** Не разбирается в технологиях. Хочет понять: «решишь мою проблему?», «сколько стоит?», «можно доверять?». Принимает решение за 1-3 минуты на сайте.

### 1.3 Позиционирование

**Кто:** Цифровой мастер — один специалист вместо целой команды.
**Что делает:** Сайты, приложения, AI-агенты — от идеи до запуска.
**Для кого:** Бизнес, который хочет расти через технологии.
**Отличие:** Полный цикл (сайт + приложение + AI) в одних руках. Без менеджеров, без испорченного телефона.

### 1.4 Технические требования

- Один HTML-файл (стили, скрипты, SVG — всё внутри)
- Никаких внешних библиотек и CDN
- Адаптивная вёрстка (mobile-first)
- Загрузка < 3 секунд
- Контраст текста WCAG AA (минимум 4.5:1)
- Поддержка `prefers-reduced-motion`

## 2. Дизайн-система

### 2.1 Настроение и стиль

**Ключевые слова:** технологичный, живой, дружелюбный, профессиональный, современный.

**Вдохновение:** Сочетание паттернов «Story Scroll Portfolio» (нарративный скролл) + «AI Builder» (тематика нейросетей) + элементы «Bento Grid» (для секции услуг).

**Принцип:** Тёмная тема со свечением. Сайт должен «жить» — тонкие анимации, но без перегруза. Каждая анимация функциональна (направляет внимание), а не декоративна.

### 2.2 Цветовая палитра

```
Фоны:
  --bg-primary:     #0a0a0b     (основной фон)
  --bg-secondary:   #111113     (чередование секций)
  --bg-card:        #16161a     (карточки, поверхности)
  --bg-card-hover:  #1c1c22     (карточка при наведении)

Текст:
  --text-primary:   #e8e8ed     (основной текст — светлый, но не белый)
  --text-secondary: #a0a0b0     (вторичный текст — описания, подписи)
  --text-muted:     #6b6b80     (третичный — даты, мета)

Акцент (оранжевый):
  --accent:         #ff8c00     (кнопки, ссылки, активные элементы)
  --accent-hover:   #ffaa33     (hover-состояние)
  --accent-glow:    rgba(255, 140, 0, 0.15)   (свечение за элементами)
  --accent-subtle:  rgba(255, 140, 0, 0.08)   (фоновые свечения)
  --accent-border:  rgba(255, 140, 0, 0.3)    (тонкие линии, разделители)

Рамки и разделители:
  --border:         rgba(255, 255, 255, 0.06)  (границы карточек)
  --border-hover:   rgba(255, 255, 255, 0.12)  (границы при наведении)

Статусы (для кейсов):
  --success:        #34d399     (рост, положительный результат)
  --info:           #60a5fa     (нейтральная информация)
```

## 3. Структура страницы

Модель AIDA: Attention → Interest → Desire → Action.
10 секций + навигация + футер.

### 3.0 Навигация (sticky header)

**Тип:** Фиксированная шапка с эффектом «матового стекла» (`backdrop-filter: blur`).

**Содержание:**
- Логотип / имя (слева)
- Ссылки: Услуги, Портфолио, Процесс, Обо мне, Цены (центр/справа)
- CTA-кнопка: **«Обсудить проект»** (справа, акцентный цвет)

**Мобильная версия:** Бургер-меню с выезжающей панелью.

**Поведение:**
- Полупрозрачная при скролле: `background: rgba(10, 10, 11, 0.7); backdrop-filter: blur(12px);`
- Плавное появление при скролле вверх, скрытие при скролле вниз
- Активная ссылка подсвечивается оранжевым (по IntersectionObserver)

**CSS:**
```css
.nav {
  position: fixed;
  top: 0;
  width: 100%;
  z-index: 100;
  background: rgba(10, 10, 11, 0.7);
  backdrop-filter: blur(12px) saturate(180%);
  -webkit-backdrop-filter: blur(12px) saturate(180%);
  border-bottom: 1px solid var(--border);
}
```

## 5. Анимации и интерактив

### 5.1 Приоритетные анимации (обязательно)

| Анимация | Где | Как |
|----------|-----|-----|
| **Fade-up при скролле** | Все секции | IntersectionObserver + CSS transition |
| **Stagger-появление** | Карточки услуг, портфолио, тарифы | transition-delay через CSS-переменные |
| **Счётчики цифр** | Секция доверия | JS: инкремент от 0 до N за 1.5сек |
| **Typing-эффект** | Hero: смена слов | CSS animation: steps() + JS для смены слов |
| **Smooth scroll** | Навигация, CTA | CSS scroll-behavior: smooth |
| **Аккордеон FAQ** | FAQ | CSS transition на max-height |
| **Hover-подъём карточек** | Карточки | transform: translateY(-4px) + glow |
| **Фокус полей формы** | Контакты | border-color + box-shadow transition |

### 5.2 Дополнительные анимации (по желанию)

| Анимация | Где | Как |
|----------|-----|-----|
| Tilt-эффект | Карточки портфолио | JS mousemove + perspective/rotate |
| Кастомный курсор | Весь сайт (десктоп) | JS: два div следуют за мышью |
| Свечение за курсором | Карточки | JS: radial-gradient следует за мышью |
| Градиентный текст | Hero-заголовок | CSS background-clip: text + animation |
| Фоновые нейроны | Hero | SVG + CSS animation (pulse, data-flow) |
| Прогресс скролла | Навигация | JS: полоска прогресса по ширине шапки |
| Параллакс декор | Фон секций | JS mousemove + data-depth |

### 5.3 Правила анимаций

1. Анимировать **только** `transform` и `opacity` (GPU, без reflow)
2. Уважать `prefers-reduced-motion: reduce` — отключать все анимации
3. Отключать cursor-эффекты на мобильных (`hover: none`)
4. Scroll-анимации — одноразовые (observer.unobserve после срабатывания)
5. Canvas-частицы (если есть): 50-80 на десктопе, 20-30 на мобильных

## 7. Тон голоса и копирайтинг

### Формула

**Профессионально + Просто + С характером**

### Правила

| Правило | Пример |
|---------|--------|
| Результат вместо процесса | Не «разрабатываю», а «создаю сайты, которые приносят заявки» |

| Конкретика вместо абстракций | Не «быстро», а «за 5 рабочих дней» |
| Короткие предложения | Максимум 15-20 слов в предложении |
| Обращение на «вы» | Дружелюбно, но с уважением |
| Без канцелярита | Не «осуществляем разработку», а «создаю» |
| Без пафоса | Не «революционные экосистемы», а «удобные инструменты» |
