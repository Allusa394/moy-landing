# Мои команды для Claude

Напиши эту команду в чате — Claude всё сделает сам.

---

## /content
Создаёт анкету для наполнения сайта контентом (файл CONTENT.md).
Адаптируется под конкретный проект.

**Когда использовать:** когда сайт готов и нужно заполнить его реальными данными —
имя, фото, проекты, цены, отзывы, контакты.

---

## /deploy
Публикует проект в интернет.

**Что делает автоматически:**
- Создаёт репозиторий на GitHub
- Заливает все файлы
- Включает GitHub Pages (сайт доступен по ссылке)
- Создаёт README.md, CLAUDE.md, PLAN.md

**Когда использовать:** когда проект готов и нужно опубликовать.

---

## Как пользоваться
1. Открой проект в VS Code
2. Напиши команду в чат Claude
3. Claude всё сделает сам
